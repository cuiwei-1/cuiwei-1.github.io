一、Java概述

  1. Java是一种面向对象的编程语言，由Sun Microsystems公司于1995年推出。
  2. Java具有跨平台性，一次编写，到处运行。
  3. Java主要应用于企业级应用、Android移动应用、Web应用等领域。
    二、Java环境配置
  4. 下载JDK（Java Development Kit）并安装。
  5. 配置环境变量：将JDK的bin目录添加到系统环境变量Path中。
  6. 验证安装：在命令行输入java -version，若显示Java版本信息，则表示安装成功。
    三、Hello World程序

  7. Java程序由类（Class）组成，每个类包含方法和属性。
  8. 程序入口为main方法，格式为：public static void main(String[] args) { }
  9. Java严格区分大小写。
  10. 每条语句以分号（;）结束。
    五、变量和数据类型
  11. 变量用于存储数据，分为基本数据类型和引用数据类型。
  12. 基本数据类型包括：整型（int、long、short、byte）、浮点型（float、double）、字符型（char）和布尔型（boolean）。
  13. 引用数据类型包括类、接口、数组等。
    六、运算符
  14. 算术运算符：+、-、*、/、%、++、–
  15. 关系运算符：>、<、==、!=、>=、<=
  16. 逻辑运算符：&&、||、!
  17. 赋值运算符：=、+=、-=、*=、/=、%=
    七、流程控制
  18. 顺序结构：程序按照顺序执行。
  19. 分支结构：if、if-else、switch-case
  20. 循环结构：for、while、do-while
    八、数组
  21. 数组用于存储同一类型的数据集合。
  22. 数组声明格式：数据类型[] 数组名 = new 数据类型[数组长度];
  23. 访问数组元素：数组名[索引];
    九、面向对象基础
  24. 类（Class）：具有相同属性和方法的对象的集合。
  25. 对象（Object）：类的实例。
  26. 属性（Field）：对象具有的特性。
  27. 方法（Method）：对象可以执行的操作。

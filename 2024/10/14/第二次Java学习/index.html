第二次Java学习

接第一次Java学习笔记，以下是第二次学习Java的一些笔记内容：

一、Java基本语法

  1. 数据类型：Java有8种基本数据类型（byte, short, int, long, float, double, char, boolean）和引用数据类型（类、接口、数组等）。

  2. 变量：在Java中，变量是存储数据的容器。声明变量的语法是：数据类型 变量名; 例如：int age;

  3. 运算符：Java提供了算术运算符（+、-、*、/、%）、关系运算符（==、!=、>、<、>=、<=）、逻辑运算符（&&、||、!）等。

  4. 控制流程：Java中的控制流程包括条件语句（if-else、switch）和循环语句（for、while、do-while）。

二、面向对象编程（OOP）进阶

  1. 类的继承：Java中的类可以继承另一个类，继承是面向对象编程的一个基本特性。语法是：class 子类 extends 父类 { ... }

  2. 方法重写：子类可以重写父类的方法，以提供特定的实现。

  3. 访问修饰符：Java提供了四种访问修饰符（public、private、protected、default），用于控制类、方法和变量的访问级别。

三、Java异常处理

  1. 异常：异常是程序执行过程中发生的错误事件，它会打断正常的指令流。

  2. try-catch语句：用于捕获并处理异常。语法是：try { ... } catch (异常类型1 e1) { ... } catch (异常类型2 e2) { ... }

  3. finally块：无论是否捕获或处理异常，finally块中的代码都会被执行。

四、Java集合框架

  1. 集合框架：Java集合框架提供了一套完整的接口和类，用于存储和管理对象集合。

  2. 常用集合类：List（ArrayList、LinkedList）、Set（HashSet、TreeSet）、Map（HashMap、TreeMap）等。
    五、Java输入输出（I/O）

  3. I/O流：Java I/O流用于处理输入输出操作，包括字节流（InputStream、OutputStream）和字符流（Reader、Writer）。

  4. 文件操作：使用File类来创建、删除、重命名文件和目录。
